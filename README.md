# Hand Distance Alert

Real-time hand detection and distance measurement using Mediapipe.  
If your hand comes too close to the camera (<20 cm), a beep sound is played using Pygame. The program can be closed by pressing `'q'` or clicking the window close button.

---

## Features
- Detects hand in real-time using Mediapipe.
- Measures approximate hand distance from the camera.
- Plays a sound alert when the hand is too close.
- Displays bounding box and distance on the video feed.
- Easy to exit using `'q'` or window close button.
